SafeFS
======

A Python module for safely working with the file system in a safe, auditable way.

(2021/08/05) CURRENTLY UNDER DEVELOPMENT. THIS VERSION IS HOLDING THE NAME ON PYPI.

Installation
------------

To install with pip, run:

    pip install safefs
